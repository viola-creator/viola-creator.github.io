Three quiet ways to add it.

Based on the existing /book listing page (Stays · Experiences · Retreats with the property + suite grid). Each variant treats the offer differently. The shared rule: small, embedded, never a banner.

Variant A

Editorial strip below the tabs

A thin band with hairline rules above and below, sitting between the Stays / Experiences / Retreats tabs and the first property. Mostly small caps with one italic phrase carrying the offer. Reads as the kind of dateline you'd see in a print magazine.

Italic note under each property name

A single italic line in terracotta sits directly under each property header (Kiyomizu, Kamo, Kyoto). The offer repeats at each decision moment without occupying any new visual space. Closest to the existing typographic system.

Editorial deal tile in the suite grid

An additional dark tile joins the Maana Kiyomizu row, sized exactly like a suite card. Uses the summer.png as a soft background image with the offer typeset on top. Looks like just another piece of the page rather than a banner.

What's the same across all three. The existing Book listing is recreated as a clean mockup. The deal element is the only thing that changes between variants.

Copy. Primary: "Three nights, your fourth is on us." Secondary: "Workshops 10% off with your stay." All three variants include both lines (variant B uses one fluid sentence).

On other surfaces. A ports directly to any page (just drop the strip under the nav). B is specific to multi-section pages with named headers. C is bespoke to a tile-grid layout but is the most visual of the three.
